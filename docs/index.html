Blood Runner, a game built in python3 made to teach school-age children how the circulatory system works

With alpha v0.1 we have made a simple, intuitive learning experience that is fun and easy to use for kids, and that is easy for teachers to set up.

Have a Mac? Great! You're ready to go. On Windows simply install python3 and you're golden!
